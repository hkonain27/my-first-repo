# Git Demo Project

## Description
This repository demonstrates basic Git and GitHub workflows, including cloning, committing, pushing, and pulling changes.

## Features
- Create and manage a GitHub repository
- Clone a repository to a local machine
- Track changes using Git
- Push updates to GitHub
- Pull updates from GitHub

## Files
- app.js → sample file for testing Git
- index.js → example JavaScript file
- README.md → project documentation
